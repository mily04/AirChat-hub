# Contributing to AirChat

Thank you for improving AirChat. Please keep changes focused, test them locally, and avoid committing generated output, dependency folders, secrets, local runtime data, or uploaded files.

## Development

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Pull Request Guidelines

- Keep each change focused on one feature or fix.
- Include a clear description of the behavior being changed.
- Run the TypeScript check before submitting.
- Do not commit `node_modules`, `dist`, `uploads`, `.env`, logs, private keys, certificates, or local database files.
- Preserve copyright, license, NOTICE, and third-party attribution files.
